# TL;DR;

安装 geektime-dl

mkdir geektime $$ cd geektime
python3 -m venv venv3 && source venv3/bin/activate
pip install -U geektime_dl

查看课程列表，获取课程 ID

geektime query

制作 Kindle 电子书

geektime ebook 49

打开电子书

open ebook/朱赟的技术管理课\[更新完毕\].mobi

这就足够了。
